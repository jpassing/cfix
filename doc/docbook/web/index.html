cfix — unit testing for C and C++ on Win32

Welcome the home of cfix, a unit testing framework for C and C++ on Windows.

Why cfix?

cfix makes writing unit tests in C/C++ a snap:

  • cfix finally makes writing unit tests in C/C++ as easy as writing tests for NUnit or JUnit
  • cfix implements strict separation between test code (written by the devloper as a DLL) and the test runner (provided by cfix)
  • Integrates well with debuggers, including Visual Studio
  • Provides both 32 bit and 64 bit support

Try cfix!

Download cfix and learn how to use it by reading the Tutorial. The download package also includes example code.
